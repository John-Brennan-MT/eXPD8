To add new colleagues please download the CSV template and complete all details. You must specify either your unique reference for each colleague or the Theia unique number. This will be used to find if the colleague already exists and update their details. If the references are not found, a new colleague will be imported. Files can be editing in excel, but must be saved as a CSV file using the CSV (Comma delimited) save option.
